# React Frontend - Event Management Platform

## How to Run the Project

1. **Clone the project**

   Open terminal and run:
   git clone https://github.com/MMuneeb1102/Event_Coordination_Client.git

2. **Go to the frontend folder**

   cd Event_Coordination_Client

3. **Install dependencies**

   npm install

4. **Configure API URL (if needed)**

   If your backend is running on a different URL, create a `.env` file in the root of the frontend folder and add:

   REACT_APP_API_URL=http://localhost:3000

5. **Start the development server**

   npm start

6. **Open in browser**

   Go to http://localhost:3000 (or the port React shows)

---

## Features

- View all events
- Create a new event
- Comment on events
- Mobile responsive layout

---

## Requirements

- Node.js installed
- Backend running on NestJS with Firebase
